# Nuam Exchange Frontend

Frontend administrativo inicial construido con React, TypeScript, Vite y React Router.

## Estructura real

- `src/main.tsx`: definición de rutas y montaje de la aplicación.
- `src/components/`: componentes compartidos de layout y páginas simples.
- `src/pages/`: pantallas iniciales de login, inicio y edición visual de calificaciones.
- `docs/frontend/`: documentación técnica y registro de prompts ejecutados.

## Scripts

```bash
npm install
npm run lint
npm run build
```

La aplicación no incluye backend, base de datos, integración con API ni autenticación real en esta etapa.
